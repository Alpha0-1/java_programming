# Java Web Development Concepts

This repository contains practical examples for learning basic Java web development concepts including:

- Servlets
- JSP Pages
- Session & Cookie Management
- REST APIs
- JSON Processing
- File Upload
- Security Filters
- MVC Architecture
- Deployment Instructions

Each file serves as a self-contained example with clear documentation and usage instructions.

## Getting Started

1. Install [Apache Tomcat](https://tomcat.apache.org/ )
2. Compile and deploy the servlets or JSPs
3. Follow individual READMEs or in-code comments for usage
